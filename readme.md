# Postback Test

Simple postback test playground.

## How it works

1. Post something to https://postback-test.herokuapp.com
2. See the content on the table (no need to refresh)

### Example

```
curl -X POST "https://postback-test.herokuapp.com" -d '{"foo":"bar"}'
```
